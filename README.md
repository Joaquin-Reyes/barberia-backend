# 💈 Sistema de Turnos para Barbería

Aplicación web completa para la gestión de turnos en una barbería, con integración de automatización vía WhatsApp.

## 🚀 Funcionalidades
- Creación y gestión de turnos
- Panel administrativo para visualizar reservas
- Confirmación automática de turnos vía WhatsApp
- Recordatorios automáticos (24h y 3h antes)
- Gestión de disponibilidad y horarios

## 🛠️ Tecnologías
- Node.js
- JavaScript
- Supabase (Base de datos)
- Railway (Deploy backend)
- API de WhatsApp

## ⚙️ Arquitectura
- Backend con lógica de turnos y validaciones
- Base de datos en Supabase
- Integración con servicios externos (WhatsApp)
